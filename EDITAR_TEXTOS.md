# Como editar textos de Raika Lab

Los textos principales estan en:

```txt
src/data/apps.ts
```

## Textos generales

Dentro de `copy.es` puedes editar:

- `heroTitle`: titulo principal.
- `heroText`: texto debajo del titulo.
- `appsTitle`: titulo de la seccion de apps.
- `labTitle` y `labText`: pantalla de laboratorio/fabrica de software.
- `philosophyTitle`, `philosophyBody`, `philosophyBodyTwo`: filosofia.
- `contactText`: texto de contacto.

## Textos de cada app

En el arreglo `apps` puedes editar cada tarjeta:

- `title`: nombre de la app.
- `category`: categoria.
- `status`: estado.
- `audience`: publico.
- `description`: descripcion principal.
- `detail`: detalle adicional dentro de la tarjeta.
- `androidHref`: link de Google Play.

## Imagenes

Las imagenes estan en:

```txt
public/assets
```

Si reemplazas una imagen, conserva el mismo nombre para no cambiar codigo.
